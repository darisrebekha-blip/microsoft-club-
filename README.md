# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) (or [oxc](https://oxc.rs) when used in [rolldown-vite](https://vite.dev/guide/rolldown)) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.



# Microsoft Club Website

A modern, responsive website for the Microsoft Club, built with **React**, **Vite**, and **Tailwind CSS**. This website showcases the club's projects, team members, and provides a contact form for visitors.

---

## 🌐 Live Demo

Check out the live website here:  
(https://darisrebekha.github.io/microsoft-club-site)

---

## 🛠️ Features

- **Responsive Design** – Works on desktop, tablet, and mobile.
- **Dark/Light Mode Toggle** – Easily switch between light and dark themes.
- **Navigation Bar** – Smooth routing using `react-router-dom`.
- **Projects Page** – Display all club projects in a professional layout.
- **Team Page** – Showcase team members with profile information.
- **Contact Page** – Contact form to reach the club.
- **Footer** – Simple, informative footer with links.
- **Tailwind Animations** – Smooth hover effects and background animations.

---

## 🧰 Technologies Used

- **React** – Frontend library for building UI components.
- **Vite** – Fast development server and build tool.
- **Tailwind CSS** – Utility-first CSS framework for styling.
- **React Router DOM** – Routing and navigation between pages.
- **React Icons** – Beautiful icons used for UI.
- **GH-Pages** – Hosting the website on GitHub Pages.
